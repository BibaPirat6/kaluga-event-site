## 🎯 О проекте и цель разработки
Этот проект — веб-сайт, разработанный мной для прохождения отборочного этапа в рамках всероссийского конкурса профессионального мастерства **«Профессионалы 2025»**.

Основная задача заключалась в оперативной и качественной верстке многостраничного интерфейса в строго ограниченные сроки. Упор был сделан на скорость реализации, адаптивность под различные устройства и точное соответствие предоставленным дизайн-макетам конкурсного задания.

## 🛠️ Какие навыки я прокачал на этом проекте
*   **Скоростная верстка в условиях дедлайна:** Научился распределять время, быстро принимать архитектурные решения и писать код без потери качества.
*   **Многостраничная структура:** Реализовал логичную систему навигации между страницами фестиваля, личного кабинета и календаря.
*   **Адаптивность (Responsive Design):** Сверстал гибкие сетки, благодаря которым интерфейс корректно отображается на мобильных устройствах, планшетах и десктопах.
*   **Интеграция медиа-контента:** Оптимизировал и подключил видео высокого разрешения и графические ассеты для демонстрации этапов конкурса.

## 📂 Структура репозитория

```text
├── image/                  # Папка с графическими ассетами и медиафайлами
├── style/                  # Папка со стилями сайта (CSS)
├── cabinet.html            # Личный кабинет участника (с анимациями)
├── calendar.html           # Календарь мероприятий и расписание
├── festival.html           # Главная страница фестиваля/конкурса
├── index.html              # Стартовая страница проекта
├── light-theme.html        # Альтернативная страница со светлой темой оформления
├── registration.html       # Страница регистрации новых участников
└── site404.html            # Кастомная страница ошибки 404 (Страница не найдена)
```
## 💻 Технологический стек
*   **HTML5** (Семантическая структура и правильная иерархия тегов для лучшего SEO)
*   **CSS3** (Flexbox и Grid Layout для построения сеток, Media Queries для полной адаптивности, CSS-анимации)
*   **Медиа-контент:** Оптимизированные видео-ролики (MP4) и графические ассеты

## 🚀 Инструкция по локальному запуску

Проект выполнен в виде статического сайта (Frontend only), поэтому не требует установки бэкенд-серверов или зависимостей.

### 1. Подготовка (Клонирование репозитория)
Откройте терминал и выполните команды:
```bash
# Клонируем проект
git clone https://github.com/BibaPirat6/kaluga-event-site.git

# Переходим в папку проекта
cd kaluga-event-site
```

Если вы работаете в Visual Studio Code:

1. Установите расширение **Live Server** (автор: *Ritwick Dey*)
   - Откройте вкладку Extensions (`Ctrl+Shift+X`)
   - В поиске введите `Live Server`
   - Нажмите **Install**
2. Откройте папку проекта в редакторе
3. Кликните правой кнопкой мыши по любому файлу `.html` (например, `festival.html`)
4. Выберите **"Open with Live Server"**
5. Сайт автоматически откроется в браузере по адресу `http://127.0.0.1:5500`
